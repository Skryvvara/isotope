## About
Isotope is a utility written in c++ to write iso images to usb devices.

## Build

```sh
mkdir build
cd build

cmake ..
make
```

## How to use

Linux / MacOS
```sh
# replace X with your disk number
./isotope os.iso /dev/diskX
```
